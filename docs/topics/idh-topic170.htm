Easy phase

 

 

 

 

 

 

Block size

In the easy phase, each block is balanced between probe and no-probe trials

Reward requires response

This time, presentation of the reward (toys or video) requires a positive response by the Participant

Pass condition

To pass the easy phase, the Participant must withhold response on 4 of the last 5 no-probe trials and respond on 4 of the last 5 probe trials. Important: this logic applies only in the "forward" direction, when we arrive at the easy phase from the training phase. Otherwise, the "bounce back" conditions apply

Max number of trials

 

Stop condition

The easy phase ends unsuccessfully if the max number of trials (e.g.: 30) has been reached without satisfying the pass condition. In this case, the experiment simply stops, because it does not bounce back to the training phase.

Pass condition (bounce back)

When we have bounced back to the easy phase from a failed test phase, the Participant must simply get 5 of 6 trials correct, probe or no-probe.

Max number of trials (bounce back)

Note that the stop condition for the bounce back case has a different max number of trials
