OPP software

 

 

 

 

When the OPP app starts, it first initializes the hardware and external libraries it needs. This can take several seconds, especially the first time the OPP is run after starting MATLAB. It should subsequently start much faster. If any part of the initialization fails, the corresponding light on the lower left of the main panel will turn red.

 

Toys

The OPP first establishes control of the MVRA toys through the Ontrak controller. (In creating this example, I don't have the Ontrak controller, and so it has been disabled.)

Video

It next launches the VLC video display.

Audio

A custom LabVIEW-based .NET library is used to send the audio signals to the MOTU USB sound card.

Mixer

Finally, it launches a gui that provides manual control of the MOTU device for adjusting sound levels and signal mixing.
