Phase values

 

 

 

 

The Phase values control the values listed in the Phase table on the OPP main panel. In this example, where we are fixing the masker level and varying the probe, these values correspond specifically to the tested probe levels. The Acoustic PTC app sets these values by default. Once they are changed in the Settings dialog, they "stick"--any subsequent measurement of the same type (i.e.: fix masker, vary probe) will use the values you enter here.
