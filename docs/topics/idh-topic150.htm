Phase settings

 

 

 

 

I have tried to distill the essence of the settings sent to us by David Horn's group to simplify the specification of the overall experiment flow. Their program allows for a much more complicated experiment structure, but for the specific case of fixed-masker-level, varied-probe-level stimulus paradigm, their settings appear to reduce to the following.

 

There are three distinct types of phase to the experiment: a training phase, an easy phase, and a series of test phases. The test phases have identical rules and differ only in the probe level. A failure on any test phase results in a "bounce back" to the easy phase. If the Participant passes the easy phase after a bounce back, I am not sure where it should go: back to the test phase where they just failed?

 

 

 

Whether or not I got the logic exactly right, note that the Investigator can manually select the phase by clicking on it in the phase table on the OPP main panel.

 

The rules for each type of phase can be inspected and modified by clicking the edit button in the corresponding row of the phase table. There is an edit button only for the first test phase because the remaining test phases follow the same rules. See the following sections (or click on the boxes in the flow chart) to see details about each phase.
