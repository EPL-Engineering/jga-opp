Testing the toys

 

 

 

 

Control of the MVRA toys can be tested by selecting Tools > Toys > Test from the main menu

 

 

This brings up the following dialog box from where the toys can be turned on and off. The toys can also be controlled using the keyboard from the OPP main panel:

 

1: toggles Toy 1

2: toggles Toy 2

3: turns both toys off
