Test phase

 

 

 

 

Block size

As in the easy phase, the test phase has blocks balanced between probe and no-probe trials

Reward requires response

Presentation of a reward (toys or video) requires a positive response from the Participant

Pass condition

As in the easy phase, successful completion of the test phase requires a withhold on 4 of the last 5 no-probe trials, and a response to 4 of the last 5 probe trials

Max number of trials

 

Bounce back condition

The test phase ends unsuccessfully and bounces back to the easy phase if a) the max number of trials (e.g.: 30) has been reached, or b) the Participant misses 3 probes trials in a row.
